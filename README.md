# Interfaz de Algoritmos Metaheurísticos

Aplicación de escritorio para visualizar y comparar algoritmos metaheurísticos
(GA, PSO, ACO, AIS, DE) sobre cuatro tipos de problemas (Función Continua,
Knapsack, Categórica Mixta, TSP).

## Requisitos

- **Python 3.10 o superior**
- Las librerías listadas en `requirements.txt` (tkinter viene incluida con Python)

> **Linux / Ubuntu:** si tkinter no está disponible, instálalo con:
> ```
> sudo apt install python3-tk
> ```

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python main.py
```
